Closed-Loop Spectral Densities

Full derivation of Suu(f) and Syu(f)
System equations
y = S0[ G0·r1 + v ]
u = S0[ r1 − C·v ]
y = G0·u + v  ,  u = r1 − C·y  ,  S0 = 1/(1+G0C)
y — plant output
u — control action
r1 — reference (WSS)
v — disturbance (WSS)
S0 — sensitivity function
G0 — plant transfer function
C — controller
r1 ⊥ v (uncorrelated, jointly WSS)
Part 1 — Deriving Suu(f)
1.1
Decompose u(t) into independent contributions
time domain
Given:
u(t) = S0(q)·r1(t) − S0(q)·C(q)·v(t)
from system equations
Identify two terms:
u = ur + uv
superposition
Reference component:
ur = S0·r1  —  transfer function from r1 to u is S0
Disturbance component:
uv = −S0C·v  —  transfer function from v to u is −S0C
1.2
Autocorrelation Ruu(τ)
time domain
Definition:
Ruu(τ) = E[ u(t)·u(t+τ) ]
Substitute u = ur + uv:
= E[ (S0r1 − S0Cv)(t) · (S0r1 − S0Cv)(t+τ) ]
expand
Expand the product:
= E[S0r1·S0r1] − E[S0r1·S0Cv] − E[S0Cv·S0r1] + E[S0Cv·S0Cv]
4 cross terms
r1 ⊥ v → cross-terms = 0:
= RS0r1(τ)  +  E[r1·v]  +  E[v·r1]  +  RS0Cv(τ)
= 0 by assumption
1.3
Apply Wiener–Khinchin → Suu(f)
freq domain
Take Fourier transform:
Suu(f) = ℱ{ RS0r1(τ) } + ℱ{ RS0Cv(τ) }
linearity of ℱ
LTI rule Syy = |H|²·Sxx:
ℱ{ RS0r1 } = |S0|² · Sr1r1(f)
H = S0
LTI rule for −S0C·v:
ℱ{ RS0Cv } = |S0|²|C|² · Svv(f)
|−S0C|² = |S0|²|C|²
★ Result — Suu(f)
Suu = |S0|² · Sr1r1  +  |S0|²|C|² · Svv
Factored form
Suu = |S0|² · [ Sr1r1  +  |C|² · Svv ]

Real-valued and non-negative. The control signal variance is the sum of: reference shaped by |S0|², and disturbance shaped by |S0C|² (the complementary sensitivity weighting on noise).

Part 2 — Deriving Syu(f)
2.1
Strategy — General Cross-Spectrum for Linear Combinations
setup
Express both signals:
y = S0G0·r1  +  S0·v
given
u = S0·r1  −  S0C·v
given
Cross-spectrum rule (r1 ⊥ v):
Syu(f) = Hyr(f) · Hur*(f) · Sr1r1(f)  +  Hyv(f) · Huv*(f) · Svv(f)
general formula for uncorrelated inputs
Identify transfer functions:
Hyr = S0G0  ,  Hyv = S0  ,  Hur = S0  ,  Huv = −S0C
2.2
Compute each cross-spectrum term
freq domain
Term from r1:
Hyr · Hur* · Sr1r1 = (S0G0) · (S0)* · Sr1r1
= |S0|² · G0 · Sr1r1
|S0|² = S0·S0*
Term from v:
Hyv · Huv* · Svv = (S0) · (−S0C)* · Svv
= − |S0|² · C* · Svv
S0·(−S0C)* = −|S0|²C*
Sum:
Syu(f) = |S0|²G0 · Sr1r1  −  |S0|²C* · Svv
★ Result — Syu(f)
Syu = |S0|² · [ G0 · Sr1r1  −  C* · Svv ]

Complex-valued in general. The two terms have opposite signs: the reference drives positive correlation between y and u (both increase together through G0), while the disturbance creates a negative correlation (C* acts to suppress v, so u opposes the effect of v on y).

2.3
Consistency check via y = G0·u + v
verification
From y = G0·u + v:
Syu(f) = G0 · Suu(f)  +  Svu(f)
cross-spectrum of linear combination
Compute Svu(f) first:
Svu(f) = Hvr · Hur* · Sr1r1 + Hvv · Huv* · Svv
Hvr = 0, Hvv = 1
= 0·S0*·Sr1r1  +  1 · (−S0C)* · Svv = −S0*C* · Svv
r1 → v path doesn't exist
Substitute Suu and Svu:
G0·Suu + Svu = G0[|S0|²Sr1r1 + |S0|²|C|²Svv] − S0*C*Svv
Group noise terms:
noise part = |S0|²G0|C|²Svv − S0*C*Svv = S0*C*Svv · (S0G0C − 1)
factor out S0*C*
Use S0G0C − 1 = −S0:
S0G0C = G0C/(1+G0C)  →  S0G0C − 1 = −1/(1+G0C) = −S0
identity from S0 definition
Therefore noise part =
S0*C*Svv·(−S0) = −|S0|²C*Svv
Final assembly:
|S0|²G0Sr1r1 − |S0|²C*Svv  ✓
matches Step 2.2 ✓
Identity used in verification
S0G0C − 1 = −S0 key identity

This follows directly from S0 = 1/(1+G0C): multiplying gives S0G0C = G0C·S0, and subtracting 1 = (1+G0C)·S0 yields −S0.

Summary
Quantity Result Properties
Suu(f) |S0|²·Sr1r1 + |S0|²|C|²·Svv Real, ≥ 0. Factored: |S0|²·[Sr1r1 + |C|²·Svv]
Syu(f) |S0|²G0·Sr1r1 − |S0|²C*·Svv Complex in general. Verified via y = G0u + v.
Suy(f) |S0|²G0*·Sr1r1 − |S0|²C·Svv Suy = Syu* (Hermitian conjugate)
S0 = 1/(1+G0C)  ·  r1 ⊥ v (jointly WSS)  ·  Key identity: S0G0C − 1 = −S0  ·  Suu real & ≥ 0  ·  Syu generally complex
